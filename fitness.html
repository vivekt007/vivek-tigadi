Fitness tracking app

Design for Fitness tracking app with Home, Activity, Map and Rewards screens.

Home and Activity screens

Home screen contains greatings for user and different kinds of workouts cards which he/she can pick from to start their excersice, all the screens have well defined text hierarchy and white spacing. Activity screen contains statistics and weekly progress which include steps, heart beat, calories, time, etc.

Reward and Map screens

User gets a reward which he/she can use to buy in app items and a map screen where the user can see live progress.
